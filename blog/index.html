PDS Interoperability Meeting 1st April 2020

March 24, 2020

Dixon Siu

A bi-weekly meeting hosted by Freezr/OwnYourData/Personium will start on April 1st. We will discuss PDS interoperability issues and try to implement the agreed solutions.

A Common Endpoint for Personal data Stores (CEPS) allows sharing apps and exchanging data. With CEPS, applications can be written once and be used with any CEPS-compliant personal data store. In this community meeting we will present the current status of this new initiative and invite others to join for creating a truly open and transparent group in fostering data exchange between Personal Data Stores.

Update #1 - Pilot project (Tally Zoo) with OwnYourData

January 30, 2020

Dixon Siu

During the winter vacation, OwnYourData's chief engineer had been working hard understanding Personium APIs and finally Tally Zoo Documentation can access Personium user's data store (Box). We achieved the followings in current implementation:

  1. Personium user authentication from TallyZoo
  2. Retrieve Personium Cell access token to access the data store
  3. Store and remove JSON files

Even though our goal (interoperability of personal data store) is still far away, MyData movement continues to drive collaborations among subject matter experts and passionate participants. Perseverance will finally lead us to our goal.

We will provide the following improvements for the next stage.

  1. Add a launcher from Personium's HomeApp
  2. Retrieve Personium Box access token to access the data store
  3. Create/Delete OData table
  4. Add/Delete OData table entry


Original news here: https://personium.io/blog/2019/12/09/Own-Your-Data-Pilot/
